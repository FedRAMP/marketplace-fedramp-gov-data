# marketplace-fedramp-gov-data
A new repository under the FedRAMP org that performs functionally similar to GSA/marketplace-fedramp-gov-data. This will serve as the dumping ground for the Auth Log 2.0 as well as our new status changes JSON, effectively decoupling package state from core information. 

## Update frequency
* FedRAMP certification data (`data.json`) will be uploaded to this repo between 2-3 AM ET daily.
* Marketplace status changelog (`fedramp-status-changelog.json`) will be uploaded between 3-4 AM ET daily.
